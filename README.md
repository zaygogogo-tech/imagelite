# ImageLite — 图片压缩 · 文字卡片

一个运行在浏览器中的多功能图片工具，双击 `index.html` 即可使用。

## 两大功能模式

页面顶部有一个 iOS 风格的分段控制器（Segmented Control），在两种模式间一键切换。

### 🖼️ 图片压缩
- 拖拽或点击上传图片（JPG / PNG / WebP）
- 选择使用场景卡片，系统自动决策压缩参数
- 左右对比预览原图与压缩效果
- 一键下载压缩后的图片

| 场景 | 自动策略 |
|------|---------|
| 📱 日常分享 | 最大宽度 2048px，WebP/JPEG，质量 82% |
| 🗜️ 极限压缩 | 最大宽度 1280px，强制 WebP，质量 60% |
| 🗃️ 高清归档 | 保持原尺寸和格式，质量 95% |

### 🔤 文字卡片（双层悬浮卡片设计）
- 输入任意文字（灵感、语录、笔记...），空白时自动渲染默认金句
- 选择 **6 种情绪调色盘**：初雪 / 燕麦 / 抹茶 / 晨星 / 陨石 / 暗黑
- 选择排版风格：现代 / 优雅 / 极客
- 实时预览卡片效果
- 一键生成 1080px 高清 PNG 并下载

| 情绪 | 外层色 | 内层色 | 适合场景 |
|------|--------|--------|---------|
| 初雪 (snow) | #EAF2F8 | #FFFFFF | 清新、通透 |
| 燕麦 (oat) | #F5F0E6 | #FCFAF8 | 温暖、复古 |
| 抹茶 (matcha) | #E8F0EA | #F4F9F5 | 自然、舒适 |
| 晨星 (morning) | #F2F2F7 | #FFFFFF | 简洁、经典 |
| 陨石 (meteor) | #8E8E93 | #E5E5EA | 科技、高级 |
| 暗黑 (dark) | #000000 | #1C1C1E | 沉浸、神秘 |

## 架构

```
index.html              ← 纯 HTML 骨架 + 分段控制器路由
├── css/style.css       ← 视觉层
└── js/
    ├── main.js         ← 主控制器（模式切换 + 事件绑定）
    ├── core/
    │   ├── intentEngine.js   ← 压缩策略引擎
    │   ├── compressor.js     ← 图片压缩算力层
    │   └── textRenderer.js   ← 文字卡片 Canvas 渲染引擎
    └── utils/
        └── domUtils.js       ← DOM 工具层
```

## 技术亮点

- **零外部依赖**：纯原生 HTML / CSS / JS
- **双层悬浮卡片渲染**：外层 Outer 色块 → 内层 Inner 圆角卡片（带 40px 高斯投影），五层绘制流水线
- **巨型引言装饰**：160px 左双引号 " 半透明叠加
- **精致页脚**：分割线 + 日期 + 品牌文字
- **Canvas 自动换行算法**：手写逐字符宽度测量 + 换行，完美支持中英文混排
- **动态排版**：< 30 字 → 64px 居中，≥ 30 字 → 48px 左对齐
- **Anti-Rollover**：压缩后体积 ≥ 原文件时自动回退
- **隐私安全**：所有处理在浏览器本地完成
